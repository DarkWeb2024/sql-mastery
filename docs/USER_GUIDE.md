# User guide

## The roadmap

The home page is a roadmap of SQL topics. Arrows show prerequisites: start on
the left and work right. Click an available topic to open its lesson. Dashed
nodes are planned topics that are not ready yet.

## Lessons

A topic page has a short summary, the theory, worked examples, a list of common
mistakes, and interview questions with answers you can expand. The Practice
button takes you to the exercises for that topic.

## Practice

Practice questions are grouped by difficulty, from beginner to expert. For each
question:

- Read the task and write your query in the editor.
- Press Run and check, or use Ctrl/Cmd + Enter.
- The app runs your query in the browser and compares the result to the correct
  answer. Row order is ignored unless the question asks you to sort.
- Stuck? Reveal hints one at a time, or show the full solution.

Solving a question for the first time awards experience points. Solving every
question in a topic marks the topic complete.

## The playground

The playground lets you run any SQL you like against the sample dataset. Click a
table in the sidebar to load a preview query, then edit and run.

## Certification exam

The exam presents randomised questions with a timer. You answer them one at a
time and cannot go back. Leaving the browser tab during the exam is recorded.
Pass and you receive a Certificate of Completion with a verification ID and a
downloadable PDF. This certificate records that you finished the exam here; it
is not an external accreditation.

## Progress

The dashboard shows your level, total experience, daily streak, topic
completion, achievements, and any certificates. Everything is stored in your
browser on this device, so using a different device or clearing site data starts
fresh. You can reset your progress from the bottom of the dashboard.
